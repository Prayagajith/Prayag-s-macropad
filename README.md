# Prayag's Macropad
### My wonderful macropad that can be used for anything like video editting, 3d modeling, and maybe even for streaming

<img width="1127" height="845" alt="Screenshot 2025-12-30 161753" src="https://github.com/user-attachments/assets/a1bb47bf-a2f5-4fc8-a46b-2762b261460a" />


BOM:
- 12x Cherry MX Switches
- 12x 1N4148 Diodes
- 1x SK6812 MINI Led
- 1x EC11 Rotary Encoder 
- 1x XIAO RP2040
- 12x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
