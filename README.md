# Prayag's Macropad
### My wonderful macropad that can be used for anything like video editting, 3d modeling, and maybe even for streaming

BOM:
- 12x Cherry MX Switches
- 12x 1N4148 Diodes
- 1x SK6812 MINI Led
- 1x EC11 Rotary Encoder 
- 1x XIAO RP2040
- 12x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
